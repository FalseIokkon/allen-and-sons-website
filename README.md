# Allen & Sons Table Tennis Club Website

Official website for Allen & Sons TTC. Built from the ground up as a modern replacement for the original site.

Contact [mypingpongclub@gmail.com](mailto:mypingpongclub@gmail.com) for questions or concerns.

View this site at [https://mypingpongclub.com](https://mypingpongclub.com).

The contact form is implemented using Google Forms, with submissions collected and automatically emailed to accounts via Google App Scripts.

The announcements feed is powered by a Google Sheet, allowing updates to be reflected on the website dynamically without modifying the code.
